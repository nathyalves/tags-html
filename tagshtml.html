O que são as tags?

(Voltar)

As tags são elementos que determinam o tipo de conteúdo que será exibido em uma página.

Tags < h1> até < h6>

(Voltar)

Possuem a função de representar cabeçalhos, sendo o maior tamanho de cabeçalho o h1 e o menor h6.

Tag < p>

(Voltar)

É utilizada para iniciar e terminar parágrafos.

Tag < mark>

(Voltar)

Serve como um marcador de texto, ou seja, destaca partes escolhidas do texto.

Tag < small>

(Voltar)

É usada para criar textos pequenos.

Tag < i>

(Voltar)

É utilizada para deixar o texto em itálico.

Tag < u>

(Voltar)

É utilizada para sublinhar o texto.

Tag < strong>

(Voltar)

É utilizada para deixar o texto em negrito.

Tag < ol>

(Voltar)

Usada para criar uma lista do tipo "ordenada" (com itens numerados).

Tag < ul>

(Voltar)

Usada para criar uma lista do tipo "não ordenada".

Tag < li>

(Voltar)

Utilizada para demarcar cada um dos itens de uma lista, seja ela ordenada ou não ordenada.

Tag < a>

(Voltar)

Faz a ligação de uma parte do texto à um link.

Tag < hr/>

(Voltar)

Cria uma linha simples para separar partes da página.

Tag < sub>

(Voltar)

É usada para escrever um texto com letras menores e posicionadas mais abaixo do que o resto do conteúdo.

Tag < sup>

(Voltar)

É utilizada para definir um texto com letras menores e escrito acima da linha normal.

Tag < blockquote>

(Voltar)

Cria uma citação.

Tag < font>

(Voltar)

Define a fonte, a cor e o tamanho do texto

Tag < del>

(Voltar)

Define um texto que foi excluido de um documento.

Tag < abbr>

(Voltar)

Define uma abreviação ou um acrônimo, como "HTML" ou "CSS".
